# Portafolio2023

Este proyecto esta basado en mi portafolio, este esta diseñado en **Angular** ademas se utilizo **boostrap**, los movimientos de los elementos utilize la herramienta **aos**, ademas utilice **@ngx-translate** para generar el cambio de idiomas en el sitio web, este sitio consta los siguientes componentes.

    - barra de navegacion
    - home
    - sobre mi
    - proyectos
    - aplicaciones
    - skills
    - contactos
    - cv

### **Barra de navegación** 
Como en la mayoría de las paginas web es necesario contar con una barra de navegación para facilitar el desplazamiento entre los componentes de nuestra pagina web, esta consta de menús los cuales nos desplazan entre la web, ademas tenemos un pequeño menú desplegable que tambien contiene enlaces a otros sitios web. cabe resaltar que decidí que los enlaces a sitios externos se ejecuten en nuevas pestañas, esto con el fin de que las personas no salgan sin desear de nuestra web.

### **Home** 
En este componente tengo un video de fondo el una pequeña foto mia y unos pequeños textos de presentación, decidi que fuese la imagen principal de presentacion en mi portafolio.

### **Sobre mi**
Este componente tiene una pequeña descripción sobre mi, alguna interfaces que manejo ademas consta de un boton el cual nos lleva al componente CV, este boton recarga este componente en el router-outlet para asi generar una nueva pagina, en esta solo se conservan los componentes de barra de navegacion, home, y contactos.

### **Proyectos**
Proyectos contiene unas cards las cuales describen alguno de los proyectos realizados por mi, al presionar en estas cards se desplegara un modal con una mayor descripcion de los proyectos.

### **Aplicaciones**
Este componente es un complemento de proyectos pero quise manejar la sección de aplicaciones en un componente aparte, este tambien constansta de cards y modals de descripciones.

### **Skills**
En este tengo diferentes iconos de herramientas y lenguales los cuales tengo conocimientos.

### **Contactos**
El elemento de contactos contiene un pequeño formulario el cual esta por si decesean comunicarse conmigo y esta enlazado a mi correo electronico, luego tengo una barra con los iconos para visitas donde se encuentran los links a mi github, linkedin y mi whatsapp.

### **CV**
En este componente decidi agregar una vista previa de mi CV tengo la vista en version español he ingles, tambien haciendo click sobre ellos puedes descargarlos.
